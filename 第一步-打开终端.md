# 第一步：打开终端

## 📋 操作步骤

### Mac用户：

1. **按快捷键**：`Command + 空格键`（同时按住）
2. **输入**：`Terminal` 或 `终端`
3. **按回车键**

终端窗口会打开。

### Windows用户：

1. **按快捷键**：`Win + R`（同时按住Win键和R键）
2. **输入**：`cmd`
3. **按回车键**

命令提示符窗口会打开。

---

## ✅ 确认终端已打开

您应该看到一个黑色或白色的窗口，里面有提示符，类似：

```
Mac示例：
Zhuanz@MacBook-Pro ~ %

Windows示例：
C:\Users\Zhuanz>
```
