# 现在执行这个命令

由于网络问题，我们使用Personal Access Token来推送代码。

---

## 📋 步骤1：创建Personal Access Token（在浏览器中）

1. **打开浏览器**，访问：
   ```
   https://github.com/settings/tokens
   ```

2. **点击 "Generate new token"** → **"Generate new token (classic)"**

3. **填写信息**：
   - **Note**：输入任意名称（如 `deployment`）
   - **Expiration**：选择 `90 days` 或 `No expiration`
   - **Scopes**：勾选 `repo`（会自动选中所有子权限）

4. **点击 "Generate token"**

5. **⚠️ 重要**：**立即复制显示的Token**（只显示一次！）

---

## 📋 步骤2：在终端中推送（使用Token）

### 方法一：直接在命令中使用Token（最简单）

在终端中执行（**将 YOUR_TOKEN 替换为您刚才复制的Token**）：

```bash
git push https://YOUR_TOKEN@github.com/414871700-web/thread-calculator.git main
```

**示例**：
如果您的Token是 `ghp_abc123def456...`，则命令是：
```bash
git push https://ghp_abc123def456@github.com/414871700-web/thread-calculator.git main
```

**⚠️ 注意**：
- 将 `YOUR_TOKEN` 替换为您的实际Token
- Token是一个很长的字符串，从 `ghp_` 开始
- 整个命令是一行，不要换行

---

### 方法二：使用认证提示（推荐，更安全）

#### 2.1 先配置Git凭据存储

在终端执行：
```bash
git config --global credential.helper osxkeychain
```

（这个命令我已经帮您执行了，如果没有错误可以跳过）

#### 2.2 执行推送命令

在终端执行：
```bash
git push -u origin main
```

#### 2.3 输入凭据

- **Username（用户名）**：输入 `414871700-web`，按回车
- **Password（密码）**：**粘贴您的Token**（不是输入密码），按回车
